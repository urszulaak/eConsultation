Necessary libs:

- curses
  pip install windows-curses

- msql-connector
  pip install mysql-connector-python

- requests
  pip install requests

Tutoral to connect db:

1. Download XAMPP https://www.apachefriends.org/pl/download.html

3. Run Apache and MySQL
